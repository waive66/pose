# 身体倾斜检测与警报功能

本项目实现了身体倾斜检测和警报功能，可以实时监测用户的身体姿态，当检测到身体倾斜超过设定阈值且持续一定时间时，会发出警告声提醒用户调整坐姿。

## 功能特点

- **实时姿态检测**：使用YOLOv8或YOLOv11姿态模型实时检测身体姿态
- **身体倾斜计算**：基于肩部和臀部关键点计算身体倾斜角度
- **智能警报机制**：当身体倾斜超过设定阈值且持续一定时间时发出警报
- **参数可调整**：支持调整倾斜角度阈值和警报时间阈值
- **可视化显示**：在视频流上实时显示倾斜角度和关键点

## 快速开始

### 方法一：使用优化版脚本（推荐）

1. 运行优化版脚本：
   ```bash
   python YOLO/tilt_alert_optimized.py
   ```

2. 程序会自动打开摄像头，开始实时监测身体姿态。

3. 当身体倾斜超过15度且持续3秒以上时，会发出警告声。

4. 按 `q` 键退出程序。

### 方法二：使用线程版脚本

1. 运行线程版脚本：
   ```bash
   python YOLO/tilt_alert_demo.py
   ```

2. 根据提示选择模型类型（YOLOv8或YOLOv11）。

3. 程序会自动打开摄像头，开始实时监测身体姿态。

4. 当身体倾斜超过15度且持续3秒以上时，会发出警告声。

5. 按 `q` 键退出程序。

## 参数调整

### 倾斜角度阈值

倾斜角度阈值决定了系统认为身体开始倾斜的角度。默认值为15度，可以根据个人情况调整：

- 在 `tilt_alert_optimized.py` 中，修改 `tilt_threshold` 变量
- 在 `tilt_alert_demo.py` 中，修改 `thread.tilt_threshold` 变量

### 警报时间阈值

警报时间阈值决定了身体倾斜持续多长时间后发出警报。默认值为3秒，可以根据个人情况调整：

- 在 `tilt_alert_optimized.py` 中，修改 `warning_time_threshold` 变量
- 在 `tilt_alert_demo.py` 中，修改 `thread.warning_time_threshold` 变量

## 技术实现

### 倾斜角度计算

系统使用肩部中点和臀部中点的连线来计算身体倾斜角度：

1. 检测人体关键点，包括左肩、右肩、左臀、右臀
2. 计算肩部中点和臀部中点
3. 计算两点连线与垂直方向的夹角
4. 取角度的绝对值作为倾斜角度

### 警报机制

1. 当检测到倾斜角度超过阈值时，开始计时
2. 如果倾斜持续时间超过设定阈值，发出警报
3. 当身体恢复正常姿态时，重置计时器

## 注意事项

1. 确保摄像头能够清晰捕捉到完整的身体轮廓
2. 避免强烈的背光或侧光，确保光线充足
3. 保持摄像头与身体的距离适中（建议1-2米）
4. 首次使用时可能需要调整参数以适应个人情况
5. 系统仅用于辅助监测，不能替代专业医疗建议

## 故障排除

### 摄像头无法打开
- 确保摄像头未被其他程序占用
- 检查摄像头驱动是否正常
- 尝试使用不同的摄像头设备

### 姿态检测不准确
- 确保光线充足
- 确保身体在摄像头视野内
- 尝试调整摄像头角度和距离
- 考虑使用更高质量的摄像头

### 警报过于频繁或不触发
- 调整倾斜角度阈值和警报时间阈值
- 确保身体姿态变化明显
- 检查模型是否正确加载
