# 坐姿健康监测系统

本系统是一个基于计算机视觉的坐姿健康监测系统，旨在帮助学生保持良好的坐姿，预防不良坐姿导致的健康问题。

## 功能特点

### 核心功能
1. **实时坐姿监测**：使用YOLOv8、YOLOv11或YOLOv8+MediaPipe模型实时检测身体姿态
2. **身体倾斜检测**：精准识别5度以上的身体倾斜
3. **智能预警机制**：当身体倾斜超过设定时间时发出警告
4. **久坐提醒**：定时提醒学生起身活动
5. **健康数据管理**：本地存储坐姿健康数据
6. **健康报告生成**：自动生成日、周、月度坐姿健康报告
7. **报告导出**：支持导出报告为文本文件

### 技术特点
1. **多模型支持**：提供多种模型选择，平衡精度和速度
2. **参数可调节**：支持自定义倾斜阈值、警告时间、久坐时长等参数
3. **隐私保护**：所有数据本地存储，不上传原始视频
4. **轻量化设计**：适配普通笔记本、平板等低算力设备
5. **鲁棒性强**：支持复杂学习场景下的姿态检测

## 系统要求

- Python 3.8+
- PySide6
- OpenCV
- NumPy
- Ultralytics YOLO
- MediaPipe (可选)

## 安装方法

1. 克隆项目到本地
2. 安装依赖：
   ```bash
   pip install PySide6 opencv-python numpy ultralytics mediapipe
   ```
3. 下载预训练模型文件到 `GUI/ptfiles/` 目录：
   - model1.pt (YOLOv8姿态模型)
   - model2.pt (YOLOv11姿态模型)

## 使用方法

1. 运行坐姿健康监测系统：
   ```bash
   python YOLO/posture_health_monitor.py
   ```

2. 系统界面说明：
   - **控制标签页**：选择模型、调整检测参数、开始/停止监测
   - **数据标签页**：查看今日数据、生成和导出健康报告
   - **关于标签页**：查看系统信息和隐私保护说明

3. 操作流程：
   - 选择合适的模型（推荐使用YOLOv8+MediaPipe）
   - 调整检测参数（倾斜阈值、警告时间等）
   - 点击"开始监测"按钮启动摄像头
   - 系统会实时显示视频流和坐姿数据
   - 当检测到不良坐姿时，系统会发出警告
   - 当久坐时间达到设定值时，系统会提醒起身活动
   - 点击"停止监测"按钮结束监测
   - 在数据标签页查看今日数据和生成健康报告

## 数据存储

系统会在 `data/` 目录下创建每日数据文件，文件名格式为 `YYYY-MM-DD.json`，包含以下信息：
- 学习总时长
- 不良坐姿次数
- 坐姿检测次数
- 不良坐姿类型分布
- 久坐记录

## 注意事项

1. 确保摄像头能够清晰捕捉到完整的身体轮廓
2. 避免强烈的背光或侧光，确保光线充足
3. 保持摄像头与身体的距离适中（建议1-2米）
4. 首次使用时可能需要调整参数以适应个人情况
5. 系统仅用于辅助监测，不能替代专业医疗建议

## 隐私保护

- 所有数据均存储在本地，不上传至任何服务器
- 系统不会保存原始视频数据，只存储分析结果
- 数据仅用于个人健康分析，不会用于其他目的
